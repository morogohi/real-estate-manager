# 부동산 통합 관리 시스템

엑셀 파일(`260105_부동산물건관리_양인석.xlsx`)의 부동산 현황을 기반으로 만든 관리 시스템입니다.

## 실행 방법 (두 가지)

1. **프로그램(권장)**: `dist\부동산관리시스템.exe` 를 더블클릭하면 독립 창으로 실행됩니다.
   데이터는 exe와 같은 폴더의 `부동산관리_데이터.json` 파일에 자동 저장됩니다.
2. **브라우저**: `index.html` 파일을 더블클릭하면 별도 설치 없이 브라우저에서 실행됩니다.
   데이터는 해당 브라우저(localStorage)에 저장됩니다.

## 주요 기능

### 1. 대시보드
- 총 보유 물건 수 / 유형별 분포 / 주택임대사업자 등록 현황
- 계약 만기 경과·임박(90일 이내) 물건 알림
- 보증보험 미가입 임대차 목록
- 임박한 할일(민원) 및 임대사업자 등록 정보

### 2. 부동산 현황
- 38개 물건 전체 목록 (유형·소유자·임대유형별 필터, 주소 검색)
- 물건 추가 / 수정 / 삭제
- 계약 만기 D-day 자동 계산
- **엑셀(CSV) 내보내기**: 현황·임대차 목록을 한글 깨짐 없이 엑셀로 저장
- **GIS / 지도·부동산 공부 열람**: 목록·상세에서 카카오맵·네이버지도·네이버부동산·부동산공시가격알리미·일사편리·정부24·위택스·홈택스로 바로 연결 (카카오 JavaScript 키 입력 시 상세 화면에 지도 미리보기)
- **연도별 공시가격(공시지가)**: 물건별로 연도별 공시가격을 기록 → 재산세·종부세 계산에 자동 활용
- **취득가액·취득일**: 양도세 계산에 자동 연동

### 3. 임대차 관리
- 임차인 성명·연락처, 계약기간, 보증금/월세, 보증보험 가입 정보 관리
- 만기일 기준 자동 정렬 및 D-day 표시
- **임대료 5% 증액 상한 계산기** (등록임대주택 증액 제한)

### 4. 세금 계산기 (2025년 기준 세율)
- **취득세**: 주택(1~3% / 다주택 중과 8·12%), 오피스텔·토지·건물(4%), 신축·상속·증여, 지방교육세·농어촌특별세 포함
- **보유세**: 재산세(공정시장가액비율·누진세율·도시지역분·지방교육세), 종합부동산세(합산배제 등록임대 반영, 1주택 12억/일반 9억 공제, 3주택 이상 중과)
  - **보유 물건 공시가격으로 인별 자동 합산**: 소유자별로 최근 공시가격을 합산하고 등록임대주택을 자동 합산배제하여 종부세를 추정
  - 물건 선택 시 해당 물건의 최근 공시가격을 자동 입력
- **양도소득세**: 보유기간별 단기세율(70%/60%), 기본세율(6~45%), 장기보유특별공제(일반/1주택), 1세대1주택 비과세(12억), 다주택 중과 옵션, 지방소득세 포함
  - 물건 선택 시 취득가액·취득일을 자동 입력

### 5. 임대소득 신고 (사업장현황신고)
- 주택임대 사업장현황신고(매년 2월)·종합소득세 신고 참고용 **추정 수입금액** 집계
- 소유자별 보증금 합계·연 월세수입·간주임대료·총 수입금액 요약
- **간주임대료 계산기**: (보증금 합계 − 3억원) × 60% × 정기예금이자율 (3주택 이상 대상)
- 엑셀(CSV) 내보내기 지원

### 6. 할일 / 민원 처리
- 엑셀 '민원처리' 시트의 12건 업무가 초기 등록되어 있음
- 할일 추가 / 완료 체크 / 삭제, 기한별 D-day 표시

## 데이터 저장

- 데이터는 브라우저(localStorage)에 자동 저장됩니다.
- 좌측 하단 **데이터 백업(JSON)** 으로 파일 백업, **백업 불러오기**로 복원할 수 있습니다.
- 다른 PC에서 사용하려면 백업 파일을 옮겨서 불러오면 됩니다.
- **초기 데이터로 재설정**을 누르면 엑셀 기준 데이터로 되돌아갑니다.

## 파일 구성

```
부동산관리시스템/
├── index.html              # 메인 화면
├── css/style.css           # 디자인
├── js/data.js              # 초기 데이터(엑셀 기반) + 저장소
├── js/tax.js               # 세금 계산 로직
├── js/app.js               # 화면 렌더링·상호작용
├── desktop.py              # 데스크톱 앱(pywebview) 진입점
├── 부동산관리시스템.spec   # PyInstaller 빌드 설정
├── build.bat               # exe 재빌드 (더블클릭)
└── dist/부동산관리시스템.exe  # 빌드된 실행 프로그램
```

## 프로그램(exe) 다시 빌드하기

화면(`index.html`, `js`, `css`)이나 초기 데이터를 수정한 뒤 프로그램에 반영하려면
`build.bat` 을 더블클릭하면 됩니다. (Python 필요, `pywebview`·`pyinstaller` 자동 설치)
완료되면 `dist\부동산관리시스템.exe` 가 새로 만들어집니다.

## 주의사항

세금 계산 결과는 2025년 기준 세율을 적용한 **예상치**이며, 세부담 상한·감면 특례 등은
반영되어 있지 않습니다. 실제 신고·납부 시 반드시 세무 전문가의 확인을 받으시기 바랍니다.
